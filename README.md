# メモ帳

## サイト概要
メモ帳アプリです。

### サイトテーマ
手軽にメモ出来ることを意識しました。

### テーマを選んだ理由
日頃自分が思い付いた事をメモ書きしたりして考えを整理している時
メモがいっぱいになって何が何だかわからなくなるので自分で使う目線で作ってみました。
手軽に軽い気持ちで利用できるサイトを作ってみようと思いました。

### ターゲットユーザ
一般ユーザ（学生、社会人、主婦等）

### 主な利用シーン
隙間時間、ちょっとした時
